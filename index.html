About your data

Everything you log lives in this browser only (using localStorage). No server, no account, no upload.

  • Persists across page reloads, browser restarts, and visits to the GitHub Pages URL.
  • Tied to this device + this browser. Phone Chrome and laptop Chrome are separate stores.
  • Clearing browser data deletes it. Same goes for "incognito" mode.
  • To use across devices: export JSON here → AirDrop / email / Drive the file → import on the other device.
  • Saving is automatic as you type — there is nothing to "upload" when you leave the page.
